The Present

I've been working on my free time on a project called Hammer. It's a simple WebSocket-based chat app that also has an API for non real-time communication needs, eg, logging in, creating accounts, etc. It's a simple project, but I'm hoping to use it as a way to learn more about web development and to build a simple, but functional, chat app. I've been thinging about colleges. I applied only to in-state institutions, including but not limited to, Indiana University, Purdue University, Purdue Fort Wayne, Purdue Polytechnic Institute Anderson, IUPUI, and Ball State University. I ended up being admitted to IUPUI, BSU, PPA. I was waitlisted at IU Bloomington, however their deadline for getting back to me is June 15th, which is after most colleges require enrollment deposits. It does suck that I got rejected from Purdue in West Lafeyette, but IUPUI is splitting up so my degree will end up saying that I "went" there. I recently decided to go to IUPUI, majoring in Motorsports Engineering. I'm excited to start school, but I'm also a bit nervous.
